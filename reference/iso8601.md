# ISO8601

Formats date in ISO8601 date time format with timezone.

## Usage

``` r
iso8601(datetime, offset)
```

## Arguments

- datetime:

  Date-time object (POSIXct class) for example made with
  [`lubridate::now()`](https://lubridate.tidyverse.org/reference/now.html).

- offset:

  Offset period for example made with
  [`lubridate::minutes()`](https://lubridate.tidyverse.org/reference/period.html).
