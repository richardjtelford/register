# Plot qrcode with access code

Plot qrcode with access code

## Usage

``` r
plot_qrcode(
  url,
  col = c("white", "black"),
  main = "title",
  code_text = "Access code",
  accesscode
)
```

## Arguments

- url:

  url to make into a QR code

- col:

  colours for the qrcode

- main:

  Plot title

- code_text:

  Access code title

- accesscode:

  accesscode
